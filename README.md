# Capella University Homepage

Vanilla HTML, CSS, and JavaScript implementation of the Capella University homepage from Figma.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

To build for production:

```bash
npm run build
npm run preview
```

## Project structure

```
index.html       # Page markup
css/
  tokens.css     # Design tokens
  styles.css     # All styles
js/
  main.js        # Carousel interactivity
```
